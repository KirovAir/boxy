# Contributing

Contributions welcome. You need the .NET 10 SDK and FFmpeg on your PATH. `dotnet run --project Boxy.Web`
runs it locally and `dotnet test` runs the tests; `dotnet build` enforces the `.editorconfig`, so
`dotnet format` handles most style nits. See the [README](README.md) for Docker and configuration.

For security issues, please use GitHub's private "Report a vulnerability" rather than a public issue.
